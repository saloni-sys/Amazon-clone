# Amazon Clone
Overview

This is a responsive Amazon Clone built using HTML and CSS. The project replicates the basic user interface of Amazon's homepage, including the navigation bar, search section, hero banner, product categories, and footer.

#Features
- Responsive Navigation Bar
- Amazon-style Search Box
- Hero Section Banner
- Product Category Boxes
- Deals and Shopping Sections
- Footer 
- Responsive Design using CSS
- Technologies Used
- HTML5
- CSS3
- Font Awesome
